# Founded Test

## Getting Started

The project is using docker, but feel free to use without it

### Prerequisites

Docker && Docker Compose

or

NODE 8


### Running
Using docker

```
docker-compose up
```

Without docker


```
cd server
```

```
npm install
```

```
npm start
```

## Testing
Using docker

```
docker-compose run server bash
npm run test
```

Without docker

```
cd server
```


```
npm run test
```
